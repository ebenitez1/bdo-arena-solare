# Class Icons

Reference storage for BDO class icon images.

## Naming convention

Lowercase, hyphens for spaces, `.png` extension:

- `lahn.png`
- `dark-knight.png`
- `kunoichi.png`
- `seraph.png` (special: Witch's alternate awakening)

## Mapping

See `../icons.json` at repo root for the canonical class → file path mapping. Future site code reads from there.
